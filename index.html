Programação

08:30 às 09:00 Recepção e Credenciamento
09:00 às 09:45 Jonhnatha Trigueiro - "Introdução ao Python para quem quer programar Web (com Django)"
Revisaremos aspectos de POO e veremos coisas úteis que o Python pode proporcionar ao dia-a-dia do desenvolvedor web. Veremos conceitos de classes, módulos, tipos de métodos e também uma introdução à arquitetura de aplicativos web modernos.
09:45 às 10:30 Diego Toral - "Interface de Administração do Django"
Uma das partes mais poderosas do Django é a sua interface automática de administração. Ela lê metadados de seus modelos para disponibilizar uma poderosa interface pronta para produção que você poderá utilizar para gerenciar as suas aplicações e o seu site. Nessa palestra, você irá aprender como fazer uso desta poderosa parte do Django, indo além e a customizando adicionando várias features.
10:30 às 11:15 Marcel Ribeiro - "Formulários em templates com Django"
Uma das partes mais importantes do Django são os Forms. Com eles é possível manipular dados, limpar, validar e tratar os dados de maneira simples e rápida. Possúi também total integração com HTML5 e seus componenetes como: select, input, checkboxes e radio button e integração com a API dos Models do Django. Desta forma o módulo Forms poderá gerar de forma automática o seu código HTML.
11:15 às 12:00 Luan Fonseca - "Como são feitos os Softwares na segunda década do século XXI"
Veremos o processo completo de criação e deploy de uma aplicação Django utilizando as metodologias de desenvolvimento ágil (TDD - Test Driven Development e CI - Continuos Integration) e ferramentas modernas como Heroku e Wercker.
14:00 às 17:00 Laboratórios - "Mini Curso de Django"
Implementar uma pequena Aplicação feita em Django, aplicando cada conceito aprendido durante a manhã.
